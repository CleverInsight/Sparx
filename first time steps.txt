Daily steps:

git status - it is used to see which files have been modified
git add . - If any new files added to repository use this command "git add ."
git commit -a -m "Write the clean description message about the current changes made it can help us to revert the changes if we got some issues"

# Only push if the working directory is clean all changes are commited
git push origin master - It will deploy the code to your github repository directly

Pull request - Its not mandiatory its used only very rarely when we want to merge with my branch
